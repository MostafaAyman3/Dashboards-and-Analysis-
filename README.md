# Project Overview  
This project provides interactive dashboards that offer insights into various domains.  

## Dashboards  
### 1. Sales Dashboard  
This dashboard visualizes sales data, helping users to track sales performance and trends over time.  

### 2. Amazon Phones Dashboard  
An analytical tool that focuses on sales and reviews of Amazon phones, allowing users to compare models based on various attributes.  

### 3. Air Quality Dashboard  
This dashboard provides analysis and tracking of air quality parameters across different regions, providing valuable insights for health and environment studies.  

## Features  
- Interactive visualizations  
- Real-time data updates  
- User-friendly interface  
- Multi-language support (Arabic and English)  

## Installation Instructions  
1. Clone the repository:  
   ```  
   git clone https://github.com/MostafaAyman3/Dashboards-and-Analysis-.git  
   ```  
2. Navigate to the project directory:  
   ```  
   cd Dashboards-and-Analysis-  
   ```  
3. Install the required dependencies:  
   ```  
   npm install  
   ```  

## Usage Guide  
After installation, you can run the dashboards using:  
```  
npm start  
```  
This will start the server and open the dashboards in your default web browser.

# المشروع  
يوفر هذا المشروع لوحات تحكم تفاعلية تقدم رؤى حول مجالات متنوعة.  

## لوحات التحكم  
### 1. لوحة تحكم المبيعات  
ت visualize هذه اللوحة بيانات المبيعات، مما يساعد المستخدمين على تتبع أداء المبيعات والاتجاهات بمرور الوقت.  

### 2. لوحة تحكم هواتف أمازون  
أداة تحليلية تركز على مبيعات ومراجعات هواتف أمازون، مما يسمح للمستخدمين بمقارنة الطرازات بناءً على معايير مختلفة.  

### 3. لوحة تحكم جودة الهواء  
تقدم هذه اللوحة تحليل وتتبع لمعايير جودة الهواء عبر مناطق مختلفة، مما يوفر رؤى قيمة لدراسات الصحة والبيئة.  

## الميزات  
- تصورات تفاعلية  
- تحديثات بيانات في الوقت الفعلي  
- واجهة مستخدم سهلة الاستخدام  
- دعم متعدد اللغات (العربية والإنجليزية)
